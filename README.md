# ab-test-chart

This is a/b test chart example interface implementation:


Implementation:

- Recharts
- Vite
- Antd

Features:
- Recharts a/b test chart
- Variations selector
- Time period selector
- Line style selector (Line, Smooth, Area)
- Light / Dark theme toggle
- Export chart to PNG
- Responsive layout

For local setup:

- need Node.js installed
- clone repository
- switch to repository directory
- run 'npm install'
- run 'npm run dev'
